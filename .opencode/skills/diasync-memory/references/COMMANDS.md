# Command Matrix

- `init`: explicit store initialization
- `sync`: instance lifecycle events
- `attach`: new-session anchor capsule
- `capture`: private stream event write
- `distill`: stream to view transformation
- `publish`: shared bus emission
- `reduce`: bus to view convergence
- `lease`: contested key ownership
- `reconcile`: superseding correction
- `checkpoint`: in-session anti-drift state update
- `handoff`: end-session continuity capsule
- `agenda`: per-project queue operations
- `hygiene`: rotate/archive/reindex maintenance
- `validate`: schema and integrity checks
- `diagnose`: health scoring and findings
- `optimize`: action planning and safe execution
- `stats`: quick operational overview
